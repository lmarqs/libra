Libra — physical devices & communication protocols

A self-balancing beam: an ESP32 at the pivot reads beam tilt from an IMU and drives two propellers to hold a target angle. Below is every device and the protocol on each link.

SELF-BALANCING BEAM
I²C— digital sensor bus, ESP32 ⇄ MPU6050
PWM— 1–2 ms servo pulse, controller → ESCs
3-phase— commutated motor drive, ESC → motor
Wi-Fi— SoftAP + HTTP, browser ⇄ ESP32
USB serial— console, laptop ⇄ ESP32
Power— battery rail behind the arm switch

Pin map (C3 / WROOM-32):  I²C SDA 2 / 32 · SCL 3 / 33  |   ESC signal 0,1 / 16,17   |   C3 = native USB-CDC · WROOM-32 = USB-UART bridge
